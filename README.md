## Instalação

1. Clone o repositório:

   ```bash
   git clone https://github.com/seu-usuario/projeto-teste.git
   ```
2. Acesse a pasta do projeto:

   ```bash
   cd projeto-teste
   ```
3. Instale as dependências:

   ```bash
   npm install  # ou pip install -r requirements.txt
   ```
